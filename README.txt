Calculadora Financeira
=====================

Arquivos:
- index.html — interface principal (HTML)
- style.css — estilos (CSS)
- script.js — lógica (JavaScript)

O que calcula:
- Ponto de Equilíbrio: unidades necessárias e receita no ponto de equilíbrio.
  Fórmula: PE(unidades) = Custos Fixos / (Preço - Custo Variável por unidade)

- Lucratividade: margem de lucro sobre a receita.
  Fórmula: Lucratividade (%) = (Lucro Líquido / Receita) × 100

- Rentabilidade (ROI): retorno sobre o capital investido.
  Fórmula: Rentabilidade (%) = (Lucro Líquido / Capital Investido) × 100

- Markup: cálculo do percentual de markup ou do preço a partir do custo e do markup desejado.
  Fórmula: Markup (%) = ((Preço - Custo) / Custo) × 100
  Preço = Custo × (1 + Markup%)

Observações:
- As fórmulas são aplicadas conforme convenções comuns; ajuste conforme sua contabilidade se necessário.
- Abra index.html no navegador para usar a aplicação.
